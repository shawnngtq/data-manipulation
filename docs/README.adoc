== Customization

To make Sphinx documentation work in Github:

. create `.nojeykll`
. create `index.html` to redirect index page to right path
. update `conf.py` extensions and theme

== Setup

[source,bash]
----
# Cleanup
make clean

# Generate rst
sphinx-apidoc -o . ../data_manipulation/

# Generate html
make html
----
